````javascript
javascript主要分为三种对象:
	1.本地对象：native Object
    	Object、function，Array,String,Boolean,Number,Error,RangeError,SyntaxError,TypeError
		ReferenceError,URIError,EvalError,Date,RegExp

	2.内置对象：Built-in Object
		Global,Math
		isNaN,undefined,parseInt,infinity NaN这些都是Global内置对象的属性，它也没有具体的对象
        本地对象和内置对象都是esma的内部对象
        
    3.宿主对象：Host Object
    	宿主对象其实就是浏览器对象，浏览器对象window(Bom)和document(Dom)
````
